# contrast.li

Image album comparison tool.

## Usage

### Setup

    $ npm i

### Run

Requires two terminal tabs. Run the `client` command in one, and `server` in the other.

    $ npm run client

    $ npm run server

### Use

Open a browser to `localhost:3000`
